## Getting Started

### Install Dependencies

```bash
pnpm install
```

### Run @qilin/app

```bash
pnpm app:dev
```

### Build And Start @qilin/app

```bash
pnpm build:dev & pnpm build:start
```

## Create New Next APP

```bash
cd ./apps
pnpm create next-app --typescript
```

change `./apps/${app-name}/package.josn -> name` and

```bash
pnpm install
```

setting `./package.josn -> script` to run app
